# 问卷星格式规范

问卷星通过文本格式自动识别题目结构，格式错误会导致题型识别失败或选项被截断。

---

## 核心结构规则

### 1. 题目之间必须空一行

```
题目一的题干
A.选项
B.选项

题目二的题干
A.选项
B.选项
```

不空行会被识别为同一题目的内容。

### 2. 题干不得换行

题干必须写在同一行，不能中途回车：

```
✓ 你在安装或配置这些工具时，花了多少时间才跑通？（Q1选D请跳过此题）

✗ 你在安装或配置这些工具时，
  花了多少时间才跑通？
```

### 3. 题干与选项之间回车换行，各选项之间回车换行

```
题干文字
A.选项一
B.选项二
C.选项三
```

### 4. 选项不得以数字开头

以数字开头的选项会被识别为新题目的题干：

```
✗ 1.非常同意
✗ 2.同意
✗ 3.一般

✓ A.非常同意
✓ B.同意
✓ C.一般
```

量表题的数字选项也必须加字母前缀或改用文字描述。

### 5. 开放题：无选项直接空一行

无选项的题目会被自动识别为文本型（开放题），不需要任何额外标记：

```
你目前最想解决的一个具体问题是什么？（选填）

下一道题的题干
```

---

## 选项编号格式

**编号后紧跟选项文字，不加空格：**

```
A.选项文字
B.选项文字
C.选项文字
```

不是：
```
A. 选项文字   ← 错误，有空格
A、选项文字   ← 错误，用顿号
（A）选项文字  ← 错误，用括号
1.选项文字    ← 错误，数字开头会被识别为题干
```

---

## 选项文字不能有空格

问卷星会把空格识别为分隔符，导致选项文字被截断。

| 错误写法 | 正确写法 |
|---------|---------|
| `10 次以上` | `10次以上` |
| `30 分钟以内` | `30分钟以内` |
| `500 元以内` | `500元以内` |
| `1-2 小时` | `1-2小时` |
| `每周 3 次以上` | `每周3次以上` |

英文专有名词内部空格可保留：`Claude Code` ✓、`Manus / Codex` ✓

---

## 题号

题号可加可不加，加了也不影响识别。建议加上方便跳转说明引用：

```
Q1.题干文字    ← 带题号
题干文字       ← 不带题号，也可以
```

---

## 跳转说明

写在题干括号内，紧跟题干，不单独成行：

```
✓ Q3.你在安装或配置这些工具时，花了多少时间才跑通？（Q1选D请跳过此题）

✗ Q3.你在安装或配置这些工具时，花了多少时间才跑通？
  （Q1选D的同学请跳过此题）   ← 单独成行会被识别为选项
```

## 完整示例

```
Q1.上完课到现在，你打开过Claude Code/Manus/Codex这类AI Agent工具几次？
A.10次以上，已是常用工具
B.4-10次，断断续续在用
C.1-3次，试了一下
D.0次，装完就没再打开

Q2.你上一次用AI Agent工具是什么时候？
A.从来没用过
B.超过2周前
C.最近2周内
D.最近3天内

Q3.你在安装或配置这些工具时，花了多少时间才跑通？（Q1选D请跳过此题）
A.没有成功跑通过
B.超过2小时
C.30分钟到2小时
D.30分钟以内，比较顺利

Q4.你用AI Agent工具尝试过哪些事情？（多选）
A.写/改代码
B.整理或分析文档/信息
C.自动化某个重复性工作
D.其他

Q5.有什么关于AI Agent的问题，是你一直想搞清楚但还没找到答案的？（选填）

Q6.你的技术背景是？
A.没有编程基础，完全不会写代码
B.了解一点，能看懂简单代码但不会写
C.会写基础代码（Python/SQL等）
D.编程是我日常工作或学习的一部分
```

---

## 粘贴前检查清单

- [ ] 题目之间有空行
- [ ] 题干在同一行，没有换行
- [ ] 选项编号后无空格（`A.` 不是 `A. `）
- [ ] 选项不以数字开头
- [ ] 数字与单位之间无空格（`10次` 不是 `10 次`）
- [ ] 跳转说明在题干括号内，不单独成行
- [ ] 多选说明 `（多选）` 紧跟题干
- [ ] 开放题无选项，直接空一行与下题分隔
